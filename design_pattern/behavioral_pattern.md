# 행위 패턴(Behavioral Pattern)

- 객체, 클래스 사이의 알고리즘이나 책임 분배에 관련된 패턴
- 한 객체가 수행할 수 없는 작업을 여러 개의 객체로 어떻게 분배하며 객체 사이의 결합도 최소화에 중점


# 패턴 종류

|패턴 이름|
|:------|
|책임 체인(Chain of responsibility)|
|커맨드(Command)|
|인터프리터(Interpreter)|
|반복자(Iterator)|
|중재자(Mediator)|
|메멘토(Memento)|
|옵저버(Observer)|
|상태(State)|
|전략(Strategy)|
|템플릿 메소드(Template method)|
|방문자(Visitor)|
